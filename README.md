# Bank Management System

## Project Overview
This is a console-based Bank Management Application developed in C++ using Object-Oriented Programming and File Handling.

## Features
- Create Bank Account
- Deposit Money
- Withdraw Money
- Display Customer Accounts
- Persistent Data Storage using File Handling

## Technologies Used
- C++
- OOP Concepts
- Binary File Handling

## How to Run

### Compile
g++ bank.cpp -o bank

### Run
./bank

## Project Structure
- bank.cpp → Main source code
- bank.dat → Stores customer records
- README.md → Project documentation
